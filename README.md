# Savita Singh - Portfolio Website

A professional portfolio website showcasing product management, analytics, and strategy expertise.

## Features

- Responsive design
- Clean, modern interface
- Multiple pages: Home, About, Experience, Skills, Contact
- Mobile-friendly layout

## Pages

- **Home**: Introduction and overview
- **About**: Background and approach
- **Experience**: Case studies and work history
- **Skills**: Technical and professional skills
- **Contact**: Contact information

## Deployment

This site is deployed on GitHub Pages at: https://savita-singh.github.io/savita-singh/

## Local Development

Simply open `index.html` in a web browser to view the site locally.

## Technologies

- HTML5
- CSS3
- Responsive Grid Layout
